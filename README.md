# Flutter ~ Auto Scroll Circular Image Slider

A Simple example of Auto Scroll Circular Image Slider for Flutter

## Features

1. Image Slider using PageView
2. Circular and infinite scroll
3. Auto Scroll
4. Indicators
5. No extra dependency needed
